# Cashier Apps
"Cashier" is a tool/utility apps created using JavaScript, SQLite(WatermelonDB), and the React-Native framework. <br><br> Download on the release page <br> https://github.com/bmf10/cashier/releases

## Features
* **App Feature**
    * Simplicity
    * 100% offline
    * No authentication
    * Install and use
